## نحوه اجرای پروژه بر سرور محلی
1. کلون کردن پروژه بر دستگاه شخصی
`git clone https://github.com/Star-Academy/Summer1401-FE-Team06`

1. نصب پکیج های مورد نیاز
`npm install`

1. اجرای پروژه 
`ng serve`
